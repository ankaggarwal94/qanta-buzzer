**Class: TossupMCEnv(gym.Env[np.ndarray, int])**

Constructor parameters:
- questions: list[MCQuestion] - dataset to sample from
- likelihood_model: LikelihoodModel - for computing belief distributions
- K: int = 4 - number of answer options
- reward_mode: str = "time_penalty" - reward computation mode
- wait_penalty: float = 0.01 - penalty per WAIT action (only in time_penalty mode)
- buzz_correct: float = 1.0 - reward for correct buzz
- buzz_incorrect: float = -0.5 - penalty for incorrect buzz
- belief_mode: str = "from_scratch" - belief computation strategy
- beta: float = 5.0 - softmax temperature for score→probability conversion
- seed: int = 13 - random seed

Constructor logic:
- Validate questions not empty, K >= 2
- Store all parameters as instance variables
- Initialize self.rng = random.Random(seed)
- Create action_space = spaces.Discrete(K + 1)  # 0=WAIT, 1..K=buzz
- Create observation_space = spaces.Box(low=-np.inf, high=np.inf, shape=(K + 6,), dtype=np.float32)
- Initialize episode state variables: question, step_idx, prev_belief, belief, terminated, truncated, _sampled_human_buzz_pos
- Set metadata = {"render_modes": []}

Instance methods:

1. reset(seed: int | None = None, options: dict | None = None) -> tuple[np.ndarray, dict]
   - If seed provided: self.rng.seed(seed) and np.random.seed(seed)
   - Sample random question: self.question = self.rng.choice(self.questions)
   - Reset state: step_idx=0, prev_belief=None, belief=uniform(K), terminated=False, truncated=False
   - Sample human buzz position: _sampled_human_buzz_pos = _sample_human_buzz(question)
   - Return (_obs(), {"qid": question.qid})

2. step(action: int) -> tuple[np.ndarray, float, bool, bool, dict]
   - Validate environment not terminated/truncated, action in action_space
   - Initialize info = {"qid": self.question.qid}, reward = 0.0
   - If action == 0 (WAIT):
     * Save prev_belief = belief.copy()
     * Compute new belief: self.belief = _compute_belief(question, step_idx)
     * Apply wait penalty if reward_mode == "time_penalty": reward -= wait_penalty
     * Increment step_idx
     * If step_idx >= total_steps (ran out of clues):
       - Forced choice: forced_idx = argmax(belief)
       - Add buzz reward: reward += _buzz_reward(question, forced_idx, step_idx-1)
       - Set truncated = True
       - Add to info: step_idx, forced_choice, forced_correct
     * Else: Add step_idx to info
   - Else (action 1..K, buzz):
     * chosen_idx = action - 1
     * last_seen = max(0, step_idx - 1)
     * reward += _buzz_reward(question, chosen_idx, last_seen)
     * Set terminated = True
     * Add to info: step_idx=last_seen, chosen_idx, correct=(chosen_idx == gold_index)
   - Return (_obs(), float(reward), terminated, truncated, info)

3. _sample_question() -> MCQuestion
   - Return self.rng.choice(self.questions)

4. _sample_human_buzz(question: MCQuestion) -> int | None
   - If question.human_buzz_positions empty: return None
   - Build positions and weights lists from human_buzz_positions tuples
   - Return self.rng.choices(positions, weights=weights, k=1)[0]
   - Note: MCQuestion doesn't have human_buzz_positions yet (Phase 1 omitted), so this always returns None. Keep implementation for future compatibility.

5. _softmax_scores(scores: np.ndarray) -> np.ndarray
   - Stabilize: stable = scores - np.max(scores)
   - Compute: probs = np.exp(beta * stable)
   - Normalize: probs / probs.sum() (handle zero denominator → uniform)
   - Return as np.float32

6. _compute_belief(question: MCQuestion, step_idx: int) -> np.ndarray
   - If belief_mode == "from_scratch":
     * Get cumulative prefix: prefix = question.cumulative_prefixes[step_idx]
     * Score: scores = likelihood_model.score(prefix, question.option_profiles)
     * Return _softmax_scores(scores)
   - If belief_mode == "sequential_bayes":
     * Extract new fragment between run_indices[step_idx-1] and run_indices[step_idx]
     * Score fragment: scores = likelihood_model.score(frag, option_profiles)
     * Bayesian update: posterior = belief * _softmax_scores(scores)
     * Normalize (handle zero → uniform)
     * Return posterior as np.float32
   - Else: raise ValueError(f"Unknown belief_mode: {belief_mode}")

7. _obs() -> np.ndarray
   - Call extract_belief_features(belief, prev_belief, step_idx, total_steps)
   - Return result (shape K+6)

8. _step_to_token_pos(step_idx: int) -> int
   - Convert step index to token position using run_indices
   - Handle edge cases (None question, out of bounds)

9. _buzz_reward(question: MCQuestion, chosen_idx: int, last_seen_step: int) -> float
   - correct = (chosen_idx == question.gold_index)
   - If reward_mode == "simple": return 1.0 if correct else -1.0
   - If reward_mode == "human_grounded":
     * token_pos = _step_to_token_pos(last_seen_step)
     * If _sampled_human_buzz_pos and token_pos > _sampled_human_buzz_pos: return 0.0 (too late)
     * Return buzz_correct if correct else buzz_incorrect
   - Default (time_penalty): return buzz_correct if correct else buzz_incorrect

10. @property total_steps -> int
    - Return len(question.run_indices) if question exists else 1
