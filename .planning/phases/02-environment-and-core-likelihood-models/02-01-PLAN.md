  - Validate belief is 1D array
  - Compute 6 derived features:

    5. progress: step_idx / max(1, total_steps)
    6. clue_idx_norm: step_idx / max(1, total_steps - 1)
  - Return np.concatenate([belief, extras]) as float32
  - Shape: (K + 6,)
