  <action>
Port qb-rl's belief feature extraction utilities. Create models/features.py containing:

1. entropy_of_distribution(prob: np.ndarray) -> float
   - Computes information-theoretic entropy: -Σ(p * log(p))
   - Clips probabilities to [1e-12, 1.0] for numerical stability
   - Returns float entropy value

2. extract_belief_features(belief, prev_belief, step_idx, total_steps) -> np.ndarray
   - Takes belief distribution (1D array of probabilities)
   - Computes 6 additional features:
     * top_p: max probability in belief
     * margin: difference between top and second probability
     * entropy: using entropy_of_distribution()
     * stability: L1 distance from prev_belief (0.0 if prev_belief is None)
     * progress: step_idx / max(1, total_steps)
     * clue_idx_norm: step_idx / max(1, total_steps - 1)
   - Returns concatenated array: [belief[K], top_p, margin, entropy, stability, progress, clue_idx_norm]
   - Validates belief is 1D array, raises ValueError otherwise
   - Uses np.float32 dtype throughout
